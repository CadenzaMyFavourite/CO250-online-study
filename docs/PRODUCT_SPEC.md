# Product specification

## Purpose

Help one University of Waterloo student study CO 250 from private local materials without depending on a cloud account or AI service.

## Core jobs

1. See the source-derived course structure and prerequisites.
2. Read definitions, assumptions, results, procedures, mistakes, and references together.
3. Recognize a problem type and choose a solving procedure.
4. Practice with progressive hints and detailed solutions.
5. Build an exam that withholds solutions until submission.
6. Help a new learner find the first missing prerequisite and enter an easy practice path.
6. Review locally stored accuracy, hint use, confidence, and mistake categories.

## Implemented screens

- Dashboard with genuine empty state and data-derived progress
- Complete 11-unit course map
- Topic reader with overview, definitions, results, procedures, mistakes, and sources
- Searchable definition/theorem/procedure library
- Recognition-first procedure library
- Topic and mixed guided practice
- Exam builder, exam session, submission, and review
- Progress, weak-topic ranking, mistake history, and reset

## Success criteria

- No fake course or progress data
- Every formal item has a visible source
- Core workflow persists after reload
- Runtime requires no network
- Search, navigation, hint reveal, solution reveal, exam submission, and progress work by keyboard
- Layout remains usable at 1440×900, a small laptop viewport, tablet, and mobile
